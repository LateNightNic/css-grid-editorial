# css-grid-editorial
A CCS grid layout for a faux-editorial piece. The CCS grid divides the layout into a columns and rows — in this case 3 columns and 4 rows. 

## About
Using grid is beneficial as it can be used across 2 dimensions / axis, compared to the one-dimensional flexbox, which is better for alignment of items. This is useful for full-page layouts, such as editorial content as your can simultaneously control the layout across both axis more precision.

## What I learned
I learned how to create content into a flowing grid structure. Specifically I learned how to debug media query's with the grid, developing a greater understanding of auto. I also learned how allow content to span multiple columns or rows.
