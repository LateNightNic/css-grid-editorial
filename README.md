# css-grid-editorial
A CCS grid layout for a faux-editorial piece. The CCS grid divides the layout into a columns and rows — in this case 3 columns and 4 rows. 

## About
Using grid is beneficial as it can be used across 2 dimensions / axis, compared to the one-dimensional flexbox, which is better for alignment of items. This is useful for full-page layouts, such as editorial content as your can control the layout with more precision simultaneously
